# 新規サービス検討 ── 第三者検証4社 競合インテリジェンス

## 概要

エンプラ／金融など非製造領域を主戦場とする第三者検証（ソフトウェアテスト・QA）4社
── **ベリサーブ・SHIFT・AGEST・バルテス** ── のサービスを棚卸しし、比較レポートをHTMLで作成。
目的は、各社が人手で提供している検証サービス・システムを **AIエージェント化** し、
その業務・売上シェアを奪うための攻略ポイントを特定すること。

## 成果物

| ファイル | 内容 |
|---|---|
| `index.html` | 比較レポート本体（自己完結・ライト/ダーク対応・図版インラインSVG） |
| `data/services.json` | 4社のサービスリスト（機械可読・80件超） |
| `figures/figures.json` | 図版のJSONプロンプト（自然言語プロンプト＋SVG描画仕様） |

## レポートの構成

1. **ヒーロー概念図** ── サービス群 → AIエージェント中核 → シェア奪取
2. **§01 4社スナップショット** ── 規模・強み・AI武器をカード比較
3. **§02 ポジショニングマップ** ── 垂直統合度 × AI/自動化度（バブル=規模）
4. **§03 カバレッジ・ヒートマップ** ── カテゴリ×4社の注力度
5. **§04 サービス詳細比較** ── 全サービス一覧（領域/AIフィルタ付き）
6. **§05 AIエージェント化 攻略戦略** ── 代替可能性レイヤーと3ステップの攻略手順
7. **§06 付録** ── 図版JSONプロンプト

## 使い方

```bash
# ブラウザで開くだけ（依存なし・CDN不使用）
open index.html
```

## メモ

- 図版は「まずJSONプロンプト化 → そこからSVG生成」の方針で作成（`figures/figures.json`）。
- 各社サイトは調査環境の通信制限で一部直接取得できず、公式ページの検索インデックス要約と
  公開情報から再構成。価格・提供時期など細部は一次情報で要確認。
- **最大の攻略ポイント＝「テスト設計」「テスト実行」**（4社共通で人手工数が集中する下流工程）。
