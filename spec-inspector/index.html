設定

設定はすべてこのブラウザの localStorage にのみ保存されます。AI補足を有効にした場合、文書は解析のために OpenAI API へ直接送信されます（それ以外の外部送信はありません）。
AIの指摘は「AI補足」タグ付きで表示され、再現性のあるスコア計算には影響しません。AI失敗時もルールベース結果はそのまま得られます。

ローカル完結・依存なし。診断はISO/IEC 25010観点、指摘はISTQB severityで分類。
