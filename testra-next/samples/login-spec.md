# ログイン機能 仕様

## 概要
本機能はユーザーがメールアドレスとパスワードでシステムにログインするための機能である。

## ログイン
- ユーザーはメールアドレスとパスワードでログインできる。
- パスワードは8文字以上32文字以下でなければならない。
- パスワードを5回連続で誤入力するとアカウントがロックされる。
- ログイン成功後はダッシュボードへ遷移する。
- 未入力の項目がある場合はエラーメッセージを表示する。

## セキュリティ
- ログインセッションは30分でタイムアウトする。
- ロックされたアカウントは管理者のみ解除できる。
- パスワードは暗号化して保存しなければならない。

## 性能
- ログイン処理は3秒以内に応答すること。

## 互換性
- Chrome / Safari / Edge の最新版で動作すること。
