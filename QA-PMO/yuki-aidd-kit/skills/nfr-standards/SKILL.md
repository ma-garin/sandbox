# NFR Standards（Yukiプロジェクト基準）

## GitHub Pages パフォーマンス予算
- 初回読み込み: 3秒以内（4G想定）
- 単一HTMLツールの目安サイズ: 500KB以内（インライン画像は避けCDN/Base64最小限）
- フォントはGoogle Fonts CDN、`font-display: swap`必須

## アクセシビリティ
- WCAG 2.1 AA準拠を目標値とする（個人PWAも含め最低限のコントラスト比4.5:1）
- タッチターゲット最小44×44px（Fold5の折りたたみ時の狭い画面幅を考慮）

## Galaxy Z Fold5 対応基準
- 折りたたみ時（狭幅）/展開時（タブレット幅）の両方でレイアウト崩れ無し
- Tailscale経由のRemote Desktop利用時は操作レイテンシを考慮し、過度なアニメーション/オートフォーカスを避ける

## 適用範囲
- 個人PWA: 上記すべて適用
- 業務PMOツール（PC専用）: パフォーマンス予算・Fold5対応は対象外、アクセシビリティのみ適用
