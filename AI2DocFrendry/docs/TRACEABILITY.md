## 3. 非機能要件 → 検証方法

| 非機能要件 | 検証方法 | 工程 |
|---|---|---|
| NFR-01 抽出率95%以上 | 検証用サンプル（各形式3件）で実測 | 総合テスト |
| NFR-02 1,000件/30分 | 性能測定 | 総合テスト |
| NFR-03 メモリ2GB以内 | 実行時メモリ計測 | 総合テスト |
| NFR-04 macOS/Win11 | 両環境でスモークテスト | 総合テスト |
| NFR-05 3ステップ | 操作手順の実測 | 受入テスト |
| NFR-06 理解容易性 | U-3（現場担当者）による確認 | 受入テスト |
| NFR-07 継続処理 | 破損ファイル混入テスト（TV-03） | 結合テスト |
| NFR-08 決定性 | 出力ハッシュ比較（TV-26） | 結合テスト |
| **NFR-09 外部通信ゼロ** | ネットワーク監視下での実行 | 総合テスト（**必須**） |
| NFR-10 ログ非出力 | ログ全文の検査（TV-37） | 結合テスト |
| NFR-11 認証情報非埋込 | 静的検査・コードレビュー | 単体〜レビュー |
| NFR-12 プラグイン性 | 新規形式追加時の差分が新規ファイルのみであること | 設計レビュー |
| NFR-13 行数・純粋関数 | 静的検査・レビュー | 実装レビュー |
| NFR-14 カバレッジ80% | 計測（scoring配下は必須） | 単体テスト |
| NFR-15 環境構築 | クリーン環境での構築確認 | 総合テスト |
