---
description: ISO 25010・ISTQB severity準拠のQAレビューを実行（qa-review-standardsスキル使用）
---

qa-review-standardsスキルに従い、対象のコード/HTMLツールをレビューします。

1. ISO/IEC 25010ユーザビリティ5サブ特性で評価
2. 検出した欠陥はISTQB severity（Critical/Major/Minor/Cosmetic）で分類
3. Evidence-only原則：実際のコード/画面/ログに基づいてのみ判定、推測は明記して除外
4. 業務提出物の場合はISO/IEC 29119-3形式のレポートにまとめる

レビュー対象のファイルパスをユーザーに確認してから実行。
