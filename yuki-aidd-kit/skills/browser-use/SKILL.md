# ブラウザ操作（Browser Use）

実際のブラウザを起動してページを操作・観察する。その場限りの確認・調査が目的で、テストコードの永続化はしない。

## 前提: Playwright MCP
`claude-code/mcp/mcp.json.template` の `playwright` エントリを有効化する（ローカルのPlaywright/Chromiumをそのまま使うため追加インストール不要）。未登録の場合はREADMEの「推奨MCP」を参照してセットアップする。

## 使い分け
| 目的 | 使うもの |
|---|---|
| その場でページを開き操作・目視確認する | このスキル（Playwright MCP） |
| 恒久的な回帰テストコードを書く | test-automation |
| 複数URLを巡回してMarkdown化・データ抽出する | firecrawl MCP |
| ライブラリの最新ドキュメントを参照する | context7 MCP |

## 実行方針
1. 操作前にゴールを1文で確認する（「何が見えたら確認完了か」）
2. 認証情報の入力はユーザー自身に行ってもらう。パスワード等をコマンド・コードに直書きしない
3. 送信・削除・購入など破壊的操作の前に必ず確認を取る
4. 確認結果はスクリーンショットと短い要約で報告する
5. 確認が終わったらセッションを閉じる（開きっぱなしはトークンを消費し続ける）

## 注意
- 個人情報・機密情報を含むページのスクショはコミットしない
- 画面が必要な場合のみ非headlessにする。通常はheadlessで十分
