## 1. V字対応（テスト設計を含む）

```
W-3 要件定義 ───────────────── W-18 受入テスト（AC-01〜09）
   │                                 ↑ W-17 受入テスト設計
W-4 基本設計 ─────────────  W-16 総合テスト（NFR）
   │                             ↑ W-15 総合テスト設計
W-6 詳細設計 ───────────  W-14 結合テスト
   │                         ↑ W-13 結合テスト設計
W-7〜12 実装 ─────────  W-12b 単体テスト
```

**v1.0の誤り**: 右側を「実行するだけ」と扱い、テスト設計工程を置いていなかった。左側の各設計工程には対応するテスト設計が必要（ISO/IEC/IEEE 29119）。
